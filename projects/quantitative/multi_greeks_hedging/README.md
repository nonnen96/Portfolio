**Quantitative analysis and hedging of exotic options on Bitcoin using futures and vanilla options**

## Description

This project was conducted during my internship at Forvis Mazars and focuses on advanced option hedging strategies for Bitcoin derivatives. The main objective is to implement and analyze **multi-greeks hedging** techniques to manage the risks of exotic options efficiently.

Key elements of the project include:

- Modeling the dynamics of Bitcoin futures and vanilla options.  
- Implementing delta and gamma hedging strategies for exotic options.  
- **Optimization of rebalancing frequency** to minimize cumulative P&L error and transaction costs.  
- Exploring potential extensions to more advanced hedging strategies.  
- Analyzing the contribution of each greek to the total hedging error.
